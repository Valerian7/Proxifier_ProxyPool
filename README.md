# Proxifier_ProxyPool
Proxifier批量添加代理服务器

- 自动检测可用的代理
- 批量添加代理服务器


## 使用说明
提供代理列表文件，每行一个
```console
usage: Proxifier_ProxyPool.py [-h] [-f FILE] [-o OUTPUT]

python3 Proxifier_ProxyPool.py -f proxy_list.txt -o test.ppx

optional arguments:
  -h, --help            show this help message and exit
  -f FILE, --file FILE  指定要打开的代理列表文件，每行一个
  -o OUTPUT, --output OUTPUT
                        输出新的配置文件名称
```
